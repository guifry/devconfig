# iTerm2 Configuration

This folder stores iTerm2 preferences for automatic syncing.

## Setup

1. Open iTerm2 → Settings → General → Preferences
2. Check "Load preferences from a custom folder or URL"
3. Set path to: `~/projects/devconfig/iterm`
4. Check "Save changes to folder when iTerm2 quits"

Your iTerm config will now sync with this repo.
